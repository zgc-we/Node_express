# node express 工程
---
## 操作步骤
```
node 安装

npm install --g express 全局安装express

npm install -g express-generator 安装express工具

express -e express_test 创建一个项目

npm install 拉取相应插件

npm install -g hotnode 热更新

mongod --dbpath "D:\Tools\mongodb\data\db"  设置数据地址

mongod --logpath "D:\Tools\mongodb\data\db\log\mongodb.log" 设置日志地址

net start mongodb、 net stop mongodb 启动关闭服务

```
